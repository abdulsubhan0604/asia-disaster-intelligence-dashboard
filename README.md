🌏 Asia Disaster Intelligence Dashboard

An interactive data project that collects real-time rainfall data and visualizes potential disaster risks across major Asian cities.

This project demonstrates how weather APIs, data processing, and geospatial visualization can be combined to create a simple disaster monitoring dashboard.

---

 🚀 Project Overview

The **Asia Disaster Intelligence Dashboard** retrieves rainfall data from the Open-Meteo Weather API and evaluates risk levels for multiple cities.  
The results are displayed on an interactive map to help visualize potential flood or heavy rainfall conditions.

The goal of this project is to explore how data pipelines can transform raw weather data into meaningful insights.

---

 📊 Features

- Real-time rainfall data collection from Open-Meteo API
- Risk classification based on precipitation levels
- Interactive geospatial map visualization
- Exportable HTML dashboard
- Multi-city monitoring across Asia

---

🛠 Technologies Used

- Python
- Requests (API integration)
- Folium (interactive maps)
- Jupyter Notebook
- Open-Meteo Weather API
